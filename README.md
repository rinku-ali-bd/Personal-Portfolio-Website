# 🌐 Rinku Ali - Personal Portfolio Website

A modern, responsive, and clean **personal portfolio website** built with **HTML, CSS, and JavaScript** to showcase my profile, skills, projects, education, and contact information.

This portfolio represents me as a **self-taught Frontend Web Developer** from Bangladesh who is passionate about web development, clean UI design, and continuous learning.

## 📌 Features

- Responsive modern portfolio design
- Clean and professional UI
- Hero section with cover/banner image
- About section with personal introduction
- Skills section with progress bars / cards
- Projects showcase section
- Education and future goals section
- Contact section with social links
- Smooth scrolling navigation
- Mobile-friendly layout
- Simple JavaScript interactivity
